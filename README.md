# GestureLab

A browser-based interactive motion studio with switchable webcam effects.

## Included modes

- Debug Tracking
- Air Drawer
- Hand Connect
- Particle Text
- X-Ray Vision

## How to use

1. Open the site.
2. Click **Start Camera**.
3. Allow camera access.
4. Switch between modes.
5. Use **Clear** to reset the current mode.

## Notes

- Best tested in Chrome.
- GitHub Pages works fine for deployment because the app is static HTML, CSS, and JavaScript.
- Theme accent colour is saved in the browser.
